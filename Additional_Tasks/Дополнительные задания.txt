1. Напишите функцию, которая принимает строку и возвращает ее в обратном порядке, используя индексацию строк.

2. Создайте список из 10 случайных чисел и напишите функцию для нахождения второго наибольшего числа в этом списке.

3. Напишите программу, которая запрашивает у пользователя имя и возраст, сохраняет эти данные в словарь и затем выводит информацию в отформатированном виде.

4. Создайте функцию, которая принимает два списка и возвращает новый список, содержащий только уникальные элементы из обоих списков.

5. Напишите программу, которая считывает текстовый файл и подсчитывает количество слов в нем, игнорируя пробелы и знаки препинания.

6. Создайте функцию, которая принимает число n и возвращает n-ное число Фибоначчи, используя рекурсию.

7. Напишите программу, которая генерирует случайный пароль заданной длины, используя буквы, цифры и специальные символы.

8. Создайте функцию, которая принимает список строк и возвращает новый список, содержащий только строки-палиндромы.

9. Напишите программу, которая имитирует простой калькулятор, используя функции для каждой операции и обработку ошибок при делении на ноль.

10. Создайте модуль с функциями для работы с геометрическими фигурами (круг, квадрат, треугольник) и напишите программу, использующую этот модуль.

11. Напишите функцию, которая принимает произвольное количество аргументов и возвращает их сумму.

12. Создайте программу, которая запрашивает у пользователя список покупок и сохраняет его в файл, а затем может загружать этот список из файла и выводить на экран.

13. Напишите функцию, которая принимает строку и возвращает словарь, где ключи - это уникальные символы, а значения - количество их появлений в строке.

14. Создайте простую игру "Угадай число" с использованием функций и обработкой пользовательского ввода.

15. Напишите программу, которая находит все простые числа в заданном диапазоне, используя функцию для проверки числа на простоту.

16. Создайте функцию-декоратор, которая измеряет время выполнения декорируемой функции и выводит результат.

17. Напишите программу, которая читает CSV файл с данными о студентах и их оценках, а затем вычисляет и выводит средний балл для каждого студента.

18. Создайте класс "Банковский счет" с методами для внесения и снятия денег, а также для проверки баланса.

19. Напишите функцию, которая принимает список чисел и возвращает новый список, содержащий только четные числа из исходного списка.

20. Создайте программу, которая симулирует игру в кости между двумя игроками, используя функции и модуль random.

21. Напишите функцию, которая принимает строку и возвращает True, если строка является действительным email-адресом, и False в противном случае.

22. Создайте программу, которая генерирует и выводит на экран таблицу умножения заданного размера, используя вложенные циклы.

23. Напишите функцию, которая принимает список списков и возвращает его плоскую версию (одномерный список).

24. Создайте простую программу для управления задачами (to-do list) с возможностью добавления, удаления и отображения задач.

25. Напишите программу, которая шифрует и дешифрует текст, используя простой шифр Цезаря, с возможностью выбора сдвига.
